# 命令系统重构：模块抽离与余额查询

将斜杠命令系统从 `chat.tsx` 中抽离为独立模块 `commands.tsx`，并新增 `/balance` 命令支持 DeepSeek 账户余额查询。

当前相关代码位于：

- `packages/web/app/commands.tsx` — 命令系统全部逻辑（类型、命令列表、hook、补全组件）
- `packages/web/app/chat.tsx` — 主聊天页面，通过 import 引用命令模块
- `packages/server/app/main.py` — `/api/balance` 后端接口
- `packages/web/next.config.ts` — `/api/balance` 的 rewrite 规则

---

## 背景与动机

命令系统（斜杠命令补全、`handleCommand`、`insertSystemMessage`、浮层 UI）在 `chat.tsx` 中占约 150 行，与聊天核心逻辑（会话管理、消息渲染、流式通信）混在一起。随着命令越来越多（新增 `/balance`），这部分代码的独立性已经很明显，适合抽离。

同时用户希望能在聊天界面直接查询 DeepSeek 账户余额，而不用去平台网页看。

## 设计思路

### 模块抽离

命令系统本质上是一个"输入拦截器"——判断输入是否以 `/` 开头，如果是则执行对应逻辑而不发给 AI。它只需要 `setMessages` 这一个外部依赖，天然适合封装为 hook。

抽离后的结构：

```
commands.tsx
├── SlashCommand 类型 + SLASH_COMMANDS 数组
├── useCommands({ setMessages }) → { handleCommand }
├── useCommandSuggestions(input) → { suggestions, showSuggestions, ... }
└── CommandSuggestions 组件（浮层 UI）
```

`chat.tsx` 只需三行引用：

```tsx
import { useCommands, useCommandSuggestions, CommandSuggestions } from './commands';

const { handleCommand } = useCommands({ setMessages });
const { suggestions, showSuggestions, selectedSuggestion, setSelectedSuggestion } = useCommandSuggestions(input);
```

### 余额查询

`/balance` 调用 DeepSeek 官方 API `https://api.deepseek.com/user/balance`，使用 config.yaml 中配置的 `api_key` 做 Bearer 认证。对于不支持余额查询的 provider（如 grok），后端返回 404 并附带提示信息。

## 具体改动

### `packages/web/app/commands.tsx`（新建）

导出三个核心 API：

**`useCommands`** — 封装所有命令处理逻辑的 hook：

```tsx
export function useCommands({ setMessages }: UseCommandsDeps) {
  const insertSystemMessage = useCallback((text: string) => { ... }, [setMessages]);
  const handleCommand = useCallback(async (raw: string): Promise<boolean> => {
    // switch(name) 分发到各命令处理器
  }, [setMessages, insertSystemMessage]);
  return { handleCommand };
}
```

**`useCommandSuggestions`** — 管理补全状态：

```tsx
export function useCommandSuggestions(input: string) {
  const suggestions = input.startsWith('/')
    ? SLASH_COMMANDS.filter(c => c.name.startsWith(input.slice(1).split(' ')[0].toLowerCase()))
    : [];
  const showSuggestions = suggestions.length > 0 && !input.slice(1).includes(' ');
  // ...
  return { suggestions, showSuggestions, selectedSuggestion, setSelectedSuggestion };
}
```

**`CommandSuggestions`** — 浮层 UI 组件，接收状态 props 渲染候选列表。

### `packages/server/app/main.py`

新增 `GET /api/balance` 端点：

```python
_BALANCE_ENDPOINTS: Dict[str, str] = {
    "deepseek": "https://api.deepseek.com/user/balance",
}

@app.get("/api/balance")
def get_balance():
    provider = _settings.llm.provider
    endpoint = _BALANCE_ENDPOINTS.get(provider)
    if not endpoint:
        raise HTTPException(status_code=404, detail=f"Provider '{provider}' 暂无可用的余额查询接口")
    # 使用 api_key 做 Bearer 认证请求余额接口
    resp = client.get(endpoint, headers={"Authorization": f"Bearer {api_key}"})
    return {"provider": provider, "data": resp.json()}
```

扩展方式：后续如果其他 provider 也有余额接口，只需在 `_BALANCE_ENDPOINTS` 字典中加一行映射。

### `packages/web/app/chat.tsx`

删除约 150 行命令相关代码，替换为 3 行 import + hook 调用。文件从 ~719 行减少到 ~560 行。
