# 网络搜索

## 技能概述

通过 Serper（Google 搜索 API）搜索网络上的最新信息，返回新闻快讯和普通搜索结果，针对中文内容做了优化。

## 使用步骤

1. **提炼搜索关键词**：将用户问题压缩为简洁的搜索短语
2. **运行搜索脚本**：执行 `scripts/search.mjs`
3. **整合结果**：基于返回内容回答用户问题，注明来源链接

## 脚本调用方式

```bash
node skills/web_search/scripts/search.mjs --query "关键词"
```

显式读取 `packages/web/.env.local` 中的 API Key：

```bash
node --env-file=packages/web/.env.local skills/web_search/scripts/search.mjs --query "关键词"
```

## 依赖的环境变量

| 变量名 | 说明 |
|--------|------|
| `SERPER_API_KEY` | Serper API Key（serper.dev） |
| `HTTPS_PROXY` | 可选，代理地址（如 `http://127.0.0.1:7890`） |

## 搜索策略

- 默认返回：3 条新闻快讯 + 5 条普通结果
- 地区设置：中国（`gl=cn`），中文界面（`hl=zh-cn`）
- 时间范围：近一个月（`tbs=qdr:m`），确保结果时效性
- 脚本会自动在关键词中注入当天日期，帮助处理"最近"、"今天"等相对时间表达

## 输出示例
查询到结果后，用以下格式展示
```
[新闻] OpenAI 发布 GPT-5（2026-03-15）
https://...

---

[1] 标题（日期）
链接
摘要...
```

找不到相关结果时，输出你所使用的关键词，并建议用户换用其他关键词重试。
