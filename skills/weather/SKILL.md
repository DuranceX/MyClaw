# 查询实时天气

## 技能概述

通过和风天气 API 查询指定城市的实时天气，返回温度、体感温度、天气状况、风向风力、湿度、能见度等信息。

## 使用步骤

1. **提取城市名**：从用户输入中识别目标城市（如"北京"、"上海"）
2. **运行查询脚本**：执行 `scripts/query_weather.mjs`
3. **格式化输出**：将 JSON 结果解析后以易读方式展示

## 脚本调用方式

```bash
# 推荐：使用 Node.js 20.6+ 内置的 --env-file，无需安装任何依赖
node --env-file=src/web/.env.local skills/weather/scripts/query_weather.mjs --city 北京
```

脚本也内置了自动 fallback：若未通过 `--env-file` 注入，会自动尝试读取 `src/web/.env.local`，所以也可以直接调用：

```bash
node skills/weather/scripts/query_weather.mjs --city 北京
```

## 依赖的环境变量

| 变量名 | 说明 |
|--------|------|
| `QWEATHER_API_KEY` | 和风天气 API Key |
| `QWEATHER_API_HOST` | 和风天气 API 域名（免费版与付费版不同） |

## 输出格式

```
🌤 北京 实时天气
─────────────────────
温度：22°C（体感 20°C）
天气：多云
风向风力：东南风 3 级
湿度：45%
降水量：0.0 mm
能见度：20 km
```

未找到城市时，告知用户重新确认城市名称。
