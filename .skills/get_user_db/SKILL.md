# 查询用户信息

## 技能概述

本技能通过运行 `scripts/query_user.py` 脚本，根据用户提供的 **ID** 或 **姓名** 查询用户信息，并将结果以结构化的方式展示给用户。

## 使用步骤

1. **判断查询类型**：确认用户想通过 ID 还是姓名查询
2. **运行查询脚本**：使用 Bash 工具执行对应命令
3. **格式化输出**：将脚本返回的 JSON 数据解析后，以易读的格式展示给用户

## 脚本调用方式

脚本路径：`skills/get_user_db/scripts/query_user.py`

```bash
# 按 ID 查询
python skills/get_user_db/scripts/query_user.py --id <用户ID>

# 按姓名查询（支持模糊匹配）
python skills/get_user_db/scripts/query_user.py --name <姓名>

# 列出所有用户
python skills/get_user_db/scripts/query_user.py --list
```

## 输出格式

查询到结果后，用以下格式展示（Markdown 表格或清单均可，根据结果数量判断）：

**单个用户时**，展示详细信息：
```
用户信息
─────────────────────
ID：3
姓名：王芳
邮箱：wangfang@example.com
部门：技术部
职位：前端工程师
入职日期：2021-06-15
状态：在职
```

**多个用户时**，展示表格：
```
共找到 2 位用户：

| ID | 姓名 | 邮箱 | 部门 | 状态 |
|----|------|------|------|------|
| 2  | 李娜 | ...  | 产品部 | 在职 |
| 7  | 李明 | ...  | 运营部 | 在职 |
```

**未找到时**，友好提示用户重新输入。

## 注意事项

- 姓名查询支持**模糊匹配**（如输入"张"会返回所有姓张的用户）
- 当前数据为模拟数据，共 10 条用户记录
- 如果用户只说"查用户信息"但没有给出具体条件，先询问是按 ID 还是姓名查询
